# Vue vs React

## 相同点

+ 组件化思想
+ 支持 Server  Side Rendering
+ 都有 Virtual DOM
+ 数据驱动视图
+ 都有支持 native 的方案
  + `weex`
  + `React Native`
+ 都有自己的构建工具
  + `vue-cli`
  + `Create React App`

## 区别

|                  | Vue                          | React                              |
| ---------------- | ---------------------------- | ---------------------------------- |
| 数据流向         | 双向数据流                   | 单向数据流                         |
| 数据变化实现原理 | 使用可变数据                 | 使用不可变数据                     |
| 组件通信         | 子传父使用事件和回调         | 使用回调函数                       |
| diff 算法        | 双向指针，边对比，边更新 DOM | 使用 diff 队列保存需要更新哪些 DOM |
