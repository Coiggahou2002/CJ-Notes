# CSS | 盒模型

> 主要区别是宽高的定义，是否包含内距和边框

## W3C 标准盒模型

`width` 和 `height` 不包括 `padding` 和 `border`

![](https://cjpark-1304138896.cos.ap-guangzhou.myqcloud.com/note_img/20211214095623.png)

## IE 盒模型

`width` 和 `height` 包含 `padding` 和 `header`
